Before: Web services existed but were never called. JSP pages used server-side rendering. Now: JSP pages make AJAX calls to web services with 'Accept: application/json' header. Flow: 1. User visits customers.jsp 2. JavaScript loads and calls loadCustomers() 3. fetch() sends request with 'Accept: application/json' 4. CustomerServlet detects JSON header and calls webService.handleGet() 5. CustomerWebService.doGet() processes request and returns JSON 6. JavaScript receives JSON and updates the UI dynamically Result: Your application is now a true distributed system using web services!
